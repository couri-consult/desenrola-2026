# Novo Desenrola Brasil — Painel Analítico

Painel HTML autocontido sobre o Novo Desenrola Brasil (MP 1.355/2026), preparado como material de apoio para análise técnica e participação em TV.

## Acesso

- Versão publicada: ver GitHub Pages do repositório (link no menu **About** ou na aba **Settings → Pages**).
- Arquivo principal: [`index.html`](index.html).

## Estrutura

- `index.html` — painel analítico (idêntico a `painel.html`, mantido como entrada padrão do GitHub Pages).
- `painel.html` — cópia espelho do painel.
- `identidade-visual/` — tokens visuais e template HTML institucional usados no projeto.
- `docs/prompt.txt` — prompt original que originou o painel.

## Conteúdo do painel

1. Visão geral & regras do programa
2. Racional econômico
3. Comparativo Desenrola 2023 × 2026
4. Endividamento das famílias (contexto macro)
5. Mecânica fiscal — como o programa "não aparece" no primário
6. Ponto de vista dos bancos
7. FGTS & poupança nacional (com contrapontos)
8. Risco moral & punição do bom pagador
9. Histórico de programas similares
10. Comentário consolidado
11. Pontos abertos para o debate
12. Fontes, glossário & notas metodológicas

## Fontes

Dados oficiais (gov.br, Banco Central, Ministério da Fazenda, Ministério do Trabalho, MEC, IFI/Senado) e análises técnicas (CNN Brasil, Infomoney, Revista Crusoé, Idec, ABRAINC, JPMorgan, BofA, Citi). Todas as referências estão linkadas inline no painel e consolidadas na seção final.

## Identidade visual

Segue o padrão institucional do projeto, definido em [`identidade-visual/CLAUDE.md`](identidade-visual/CLAUDE.md): paleta sóbria sobre fundo branco, tipografia sans-serif, layout com nav lateral fixo, KPI cards e callouts semânticos.
